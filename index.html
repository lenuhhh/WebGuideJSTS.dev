1 Змінні та типи даних

Змінна — це «коробка», в якій зберігається значення. У JS є три способи оголосити змінну: var, let и const.

💡

Правило початківця: використовуй const за замовчуванням. Якщо значення потрібно змінювати — використовуй let. Забудь про var — це застарілий спосіб.

JavaScript
// const — значення не можна перепризначити// const — значение нельзя переназначить// const — value cannot be reassigned
const name = "Олексій""Алексей""Alexey";
const age = 25;
const isStudent = true;

// let — значення можна змінювати// let — значение можно изменять// let — value can be changed
let score = 0;
score = 10; // ✅ можно// ✅ можно// ✅ OK

// const з об'єктами та масивами — ОК!// const с объектами и массивами — ОК!// const with objects and arrays — OK!
// Сам об'єкт не можна замінити, але властивості можна змінювати// Сам объект нельзя заменить, но свойства можно менять// The object itself can't be replaced, but properties can be changed
const user = { name: "Маша""Маша""Masha" };
user.name = "Катя""Катя""Katya"; // ✅ можна змінити властивість// ✅ можно изменить свойство// ✅ can change property

5 Функції

Функція — це блок коду з іменем, який можна викликати скільки завгодно разів. Це основа повторного використання коду.

Оголошення функції (Function Declaration)
// Оголошення — піднімається вгору (hoisting)// Объявление — поднимается вверх (hoisting)// Declaration — hoisted to the top
function greet(name) {
  return `Привіт, ${name}!``Привет, ${name}!``Hello, ${name}!`;
}

console.log(greet("Маша""Маша""Masha")); // → "Привіт, Маша!"

// Кілька параметрів// Несколько параметров// Multiple parameters
function add(a, b) {
  return a + b;
}
add(3, 5); // → 8

// Параметри за замовчуванням// Параметры по умолчанию// Default parameters
function power(base, exp = 2) {
  return base ** exp;
}
power(3);     // → 9  (exp = 2 за замовчуванням)// → 9  (exp = 2 по умолчанию)// → 9  (exp = 2 by default)
power(2, 10); // → 1024
Стрілкові функції (Arrow Functions)
// Звичайна функція// Обычная функция// Regular function
const double = function(x) {
  return x * 2;
};

// Стрілкова — коротше// Стрелочная — короче// Arrow — shorter
const double = (x) => {
  return x * 2;
};

// Ще коротше — якщо один рядок, return можна прибрати// Ещё короче — если одна строка, return можно убрать// Even shorter — single expression, return can be omitted

// Кілька параметрів — дужки обов'язкові// Несколько параметров — скобки обязательны// Multiple params — parentheses required

// Без параметрів// Без параметров// No parameters
const sayHi = () => "Привіт!""Привет!""Hello!";

// Повернення об'єкта — потрібні додаткові дужки// Возврат объекта — нужны дополнительные скобки// Returning object — extra parentheses needed

makeUser("Петро""Пётр""Petro", 25); // → { name: "Петро", age: 25 }
Функції як значення — callback
// Функція — це просто значення. Її можна передавати в іншу функцію!// Функция — это просто значение. Её можно передавать в другую функцию!// A function is just a value. It can be passed to another function!

function range(from, to) {
  return {
    [Symbol.iterator]() {
      let current = from;
      return {
        next() {
          if (current <= to) {
            return { value: current++, done: false };
          }
          return { value: undefined, done: true };
        }
      };
    }
  };
}
